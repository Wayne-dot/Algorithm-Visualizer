Nonograms algorithm

Rule of Nonograms

The goal of the nonogram is to determine whether or not if the color of each individual box should be white or black. There are hits that you can find that help you to solve the game given at the left end of the column and rows. For example, if the tops box indicate the number 3, that mean in this row/column there should be a 3 consecutive black box without any white box between them.

In this project, I have code a algorithm that will solve this nongram project.
